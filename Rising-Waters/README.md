# 🌊 Rising Water – AI Flood Prediction System

## 📌 Project Overview

Rising Water is a Machine Learning-based Flood Prediction System that predicts the likelihood of flood occurrence using historical weather and environmental data. The application helps authorities and disaster management teams make informed decisions by providing early flood risk predictions through a user-friendly Flask web application.

---

## 🚀 Features

- 🌧️ Predict flood risk using Machine Learning
- 📊 Interactive and user-friendly web interface
- 📈 Live weather parameter visualization
- 🤖 Multiple ML algorithms comparison
- 💾 Saved trained model for fast predictions
- 🌐 Flask-based web application
- 📱 Responsive design

## 📊 Dataset

The dataset contains weather and environmental parameters used for flood prediction.

Features include:

- Monsoon Intensity
- Topography Drainage
- River Management
- Deforestation
- Urbanization
- Climate Change
- Dams Quality
- Siltation
- Agricultural Practices
- Encroachments
- Drainage Systems
- Coastal Vulnerability
- Landslides
- Watersheds
- Population Score
- Wetland Loss
- Political Factors

Target Variable:

- Flood (0 = No Flood, 1 = Flood)

---

## 🤖 Machine Learning Models

The following algorithms were implemented and compared:

- Decision Tree
- Random Forest
- K-Nearest Neighbors
- XGBoost

The best-performing model was saved for deployment using Joblib.

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/Navyareddy2005/Rising-Waters.git
```

Go to project folder

```bash
cd Rising_Water
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```

---

## 🎯 Prediction Process

1. Open the application.
2. Enter all weather parameters.
3. Click **Predict Flood**.
4. The model analyzes the data.
5. Flood risk is displayed.
6. Weather parameter graph is shown.
